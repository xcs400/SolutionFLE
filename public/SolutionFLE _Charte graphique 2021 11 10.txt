Dans la palette de couleurs Word, en choisissant curseur RVB, on peut modifier et personnaliser les couleurs avec le code Red /Green/Blue  (Rouge / Vert /Bleu en FR)  du tableau ci-dessus.

Les couleurs sont enregistrées dans le Thème Office SolutionFLE. Sur Mac elles se modifient en passant par PowerPoint 2011, grâce à la fenêtre ci-dessous, le thème est ensuite disponible dans Word. Un thème contient, le jeu de couleurs (palette) le jeu de police et le jeu de styles.
2.2	Le jeu de polices
Le jeu de polices contient une police de titre et une police de corps de texte pour les caractères latins et la même chose pour les autres alphabets (chinois, arabe, cyrillique…).
	Tahoma pour les titres
	Calibri pour le corps de texte
